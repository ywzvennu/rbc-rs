# Code of Conduct

This project follows the [Contributor Covenant v2.1][cc] code of conduct.
Maintainers and contributors are expected to uphold its standards in all
project spaces: issues, pull requests, code reviews, and any other
communication channel associated with the project.

The full text is available at the link below.

## Reporting

Concerns about conduct in this project can be reported to the
repository owner via GitHub by opening a private security advisory or
by email at the address listed on the maintainer's GitHub profile.
Reports are handled confidentially.

## Enforcement

Project maintainers are responsible for clarifying and enforcing the
standards described in the Contributor Covenant. Maintainers have the
right to remove, edit, or reject comments, commits, code, issues, and
other contributions that don't align with this Code of Conduct, and
may temporarily or permanently ban any contributor for behavior they
deem inappropriate.
